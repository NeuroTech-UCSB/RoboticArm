
Use commands.cpp for push, pull, open, right, left, and close commands

Use servo_test.cpp to commanding exact angles to each servo. 
Open up the serial monitor and type in <servo> <angle>
- <servo> - 1 - 4
- <angle> - 0 - 180

It's recommended that you command a servo to 90 degrees, then position the arm to the middle of the range of motion you would like. Pay attention to the direction the servo rotates when you increase the angle.
